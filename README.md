# Heart Disease Prediction using a Machine Learning Model (Year 2 Semester 1 – Group Project)

## Overview
This project predicts the risk of heart disease using machine learning models. It includes data preprocessing, model training, evaluation, and a comparison of model performance to identify the optimum model for prediction.

## Tools & Technologies
- **Python** – programming language  
- **Jupyter Notebook / VS Code** – development environment  
- **Scikit-learn** – ML model implementation (Decision Tree, Random Forest, KNN, SVM, Logistic Regression)  
- **Pandas & NumPy** – data manipulation and preprocessing  
- **Matplotlib & Seaborn** – visualization of results  

## Project Contents
- **Final Report:** [final_report.pdf](./final_report.pdf) – main project report  
- **Model Comparison Report:** [model_comparison_report.pdf](./model_comparison_report.pdf) – details of models used and performance comparison  
- **Code:** All Python scripts are in the `code/` folder  
- **Datasets:** Optional CSV files used for model training and evaluation
